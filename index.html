Переменные
Переменная – это это именованные контейнеры для хранения данных. Для создания используют const(для неизменяемых значений), let, var (устар.)
Типы данных
Тип данных — это набор характеристик значения, который отличает поведение одного типа значения от другого, JS динамически типизирован и насчитывает 8 типов: Undefined, Boolean, Number, String, BigInt, Symbol, Null, Object
Преобразование типов
Преобразование типов — это процесс конвертации значения из одного типа в другой. В JS типы можно преобразовывать явно и неявно. Различают строковое, численное и логическое преобразования.
Базовые операторы
Практически все математические операторы используются в языке в разных комбинациях для достижения определённого эффекта, оператор "+" помимо сложения чисел выполняет конкатенацию строк, "%" находит остаток от деления, а сочетание "**" возводит в степень
Операторы сравнения
Операторы сравнения значений "==" и "===", в первом случае мы сравниваем значения без учтения типа, во втором тип учитывается, т.е. "2" == 2 (true), "2" === 2 (false)
Условные операторы
Условный(тернарный) оператор "?" используется для сокращенной записи инструкции if
Логические операторы
Для сложных условий конструкции if используются 3 логических оператора || (или), && (и), !(не)
while / for
Для повторения блока кода используются циклы, к базовым относят:
while(условие) {
// ... тело цикла ...
}
 for (начало; условие; шаг) {
// ... тело цикла ...
}
switch
Когда действия выполняются в зависимости от значения переменной используют switch
switch(x) {
case 'value1':
...
[break]
case 'value2':
...
[break]
default:
...
[break]
}
Функции
Для многократного использования блока команд в разных участках программы используются функции. Функция может принимать параметры, обладает своей областью видимости, может возвращать результат выполнения. function hello(name) { alert(`Это базовый вид объявления функции`) }

JSON
JavaScript Object Notation, формат для обмена данными между браузером и сервером, есть набор пар ключ значение, тем самым похож на объект

Map и Set
Map — это коллекция пар ключ/значение, которая может использовать любой тип данных в качестве ключа и поддерживает порядок своих записей
Set — это множество уникальных значений
Коллекции Map и Set имеют свои, отличные от объектов и массивов, методы и свойства

Методы объектов
У объектов есть методы(меньше, чем у массивов), которые помогают модифицировать и итерировать их, например:
Object.create() - позволяет создавать новые объекты

Object.keys() - создает массив ключей объекта

Object.values() - создает массив значений объекта

Object.entries() - создает вложенный массив пар «ключ-значение» объекта

Object.assign() - копирует значения из одного объекта в другой

Object.freeze() - предотвращает модификацию свойств и значений объекта

Object.seal() - предотвращает добавление новых свойств объекта, но позволяет изменять существующие свойства

Object.getPrototypeOf() - используется для получения внутреннего скрытого [[Prototype]] объекта
Деструктуризация
Деструктуризация – синтаксическая возможность разложить элементы объекта(в широком мысле) в отдельные значения/переменные
Объект DATE
Объект Date позволяет работать с датами и временем с помощью встроенных методов и констант
